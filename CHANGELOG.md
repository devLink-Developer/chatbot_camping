# Changelog

## [1.0.0] - 2024-01-29

### Agregado
- ✅ API FastAPI con webhook para WhatsApp
- ✅ Base de datos PostgreSQL con modelos SQLAlchemy
- ✅ Validación robusta de entrada de usuario
- ✅ Gestión de sesiones con timeouts
- ✅ Integración con WhatsApp API
- ✅ Docker y docker-compose para containerización
- ✅ Script de importación de datos desde MongoDB
- ✅ Logging completo de interacciones
- ✅ API REST para gestión de sesiones

### Mejoras respecto a n8n
- Validación de entrada centralizada
- Manejo eficiente de emojis y Unicode
- Mejor gestión de estado de sesión
- Escalabilidad mejorada
- Facilidad de mantenimiento y extensión
- Mejor rendimiento
